# Credit_card_financial_dashboard
Power BI dashboard 
Project Objective : To develop a comprehensive creditcard weekly dashboard that provides real-time insights into key performance metrics and trends, enabling stakeholders to monitor
and analyze credit card operations effectively. 

Steps taken : Used Excel, Sql and Power BI to process and import the dataset. Applied DAX Queries and built the interactive Dashboard with various charts,tables and KPIs.
Credit card customer dashboard and credit card transactions dashboard. 

Insights : Built a ppt for for credit card financial report with all the insights.
